## Chessington (WIP)

### GUI

- Put piece SVGs/PNGs in `assets/pieces/` (see `assets/pieces/README.md`).
- Convert SVGs to PNGs (optional, but recommended): `python3 scripts/convert_svgs_to_pngs.py assets/pieces`
- Run the UI: `python3 gui.py`
  - Type moves in UCI (e.g. `e2e4`, `g8f6`, `e7e8q`) and press Enter.
